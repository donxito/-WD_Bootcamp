

# How to add style to React apps and components


There's many ways and patterns to add style to our React components.

Here's some examples:
- Inline CSS
- Normal CSS ("plain css")
  - optional: with one file with component
- CSS-in-JS
- Styled Components
- CSS modules
- Sass & SCSS
